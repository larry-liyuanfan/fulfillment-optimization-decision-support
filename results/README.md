# Results Guide

This folder contains all generated experiment outputs.

## Subfolders

- [`instances`](instances): generated scenario inputs exported as CSV and JSON
- [`tables`](tables): KPI tables, diagnostics, validation checks, and replication outputs
- [`figures`](figures): charts used for portfolio analysis and visual review

## Most Useful Files

- [`tables/experiment_summary.csv`](tables/experiment_summary.csv): main KPI table
- [`tables/model_comparison.csv`](tables/model_comparison.csv): integrated-versus-benchmark comparison
- [`tables/validation_summary.csv`](tables/validation_summary.csv): feasibility checks
- [`tables/replication_summary.csv`](tables/replication_summary.csv): replication-study summary
- [`figures/experiment_kpi_comparison.png`](figures/experiment_kpi_comparison.png): main comparison chart
